# Getting Started

DeviceScript has a development cycle tailored for TypeScript developers, node or web.

## Visual Studio (VS) Code extension

The DeviceScript extension for VS Code provides the best developer experience, with building, deploying, debugging,
tracing, device monitoring and other developer productivity features.

-   [Getting started with VS Code](/getting-started/vscode)

## Command Line Interface (CLI)

The CLI is IDE agnostic and will let you script your own developer experience.

-   [Getting started with the CLI](/getting-started/cli)
